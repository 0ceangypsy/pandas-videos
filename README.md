# Python pandas Q&A video series

This repository contains all the materials related to the Data School video series about Python's [pandas library](http://pandas.pydata.org/).

## Videos (YouTube playlist)

1. What is pandas? (Introduction to the Q&A series) (6:24)
2. How do I read a tabular data file into pandas? (8:54)

## IPython/Jupyter Notebook

This [notebook](pandas.ipynb) contains the code and resource links from every video in the series.
